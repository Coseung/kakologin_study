# Spring Boot OAuth 2.0 소셜 로그인 공부 📒


---

## 주요 기능 🔧

- **OAuth 2.0을 활용한 카카오 소셜 로그인** 🔐
- **Spring Security를 활용한 보안 설정** 🛡️
- **JWT를 사용한 인증 토큰 관리 추후 공부 예정** 🔑
